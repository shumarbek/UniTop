# Roadmap

## MVP Roadmap

1. Telegram bot start flow, Uzbek copy, Mini App launch, referral payload capture.
2. Telegram init-data verification and Supabase user upsert.
3. Terms and Privacy acceptance gate.
4. Official channel membership check.
5. Profile creation with photo upload, location, bio, interests, looking-for gender, and purpose.
6. Automatic bio/contact validation and basic photo moderation integration.
7. Admin moderation queue with approve, reject, and ban.
8. Discovery feed with daily limits and district/region ranking.
9. Like, skip, reciprocal match creation, match list, unmatch.
10. Telegram notifications for profile approval/rejection and new matches.
11. Support tickets and FAQ.
12. Admin dashboard metrics.

## Production Roadmap

1. Harden rate limits, spam detection, report workflows, and ban appeals.
2. Add payment provider integration for Uzbekistan and verified webhook handling.
3. Launch premium subscriptions and boosts.
4. Add received-likes premium screen.
5. Add referral reward automation and fraud detection.
6. Add neighboring-region ranking map for all Uzbekistan regions.
7. Improve photo moderation with face visibility and single-person confidence thresholds.
8. Add notification retry queue and scheduled premium-expiry reminders.
9. Add analytics dashboards for retention, conversion, moderation load, and revenue.
10. Add automated database backups, error monitoring, and incident runbooks.
11. Localize Terms, Privacy, support macros, and rejection reasons in polished Uzbek.
12. Security review before public launch.
